# Voxel Craft

Браузерная voxel-игра в духе Minecraft на JavaScript и Three.js.

## Текущие возможности

- Камера от первого лица (Pointer Lock)
- Движение WASD, прыжок, бег
- Простой мир из блоков с гравитацией и базовыми коллизиями

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173). Кликните по экрану, чтобы захватить мышь и начать играть.

## Сборка

```bash
npm run build
npm run preview
```
